bash on ubuntu安装php开发环境

作者: Roc xu 日期: 2017年01月06日  4:19:09 分类: 全部文章 访问量:次

1.进入root权限

sudo su

2.安装php7.1

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:ondrej/php
sudo apt-get update
sudo apt-get install php7.1

3.查询可安装的php模块

apt-cache search php7-*

4.安装nginx

sudo apt-add-repository ppa:nginx/development
sudo apt-get update
sudo apt-get install nginx

5.修改nginx配置

修改vim /etc/nginx/sites-available/default文件
修改内容如下：（修改监听的端口号并注释掉ipv6的监听）
listen 8080 default_server;#不需要修改
#listen [::]:80 default_server;

修改 /etc/nginx/nginx.conf， 添加配置(添加到第四行)
master_process off;

6.卸载nginx

注意：以下卸载方法我并没有验证过

卸载方法1.

删除nginx，保留配置文件

sudo apt-get remove nginx
#删除配置文件
rm -rf /etc/nginx
卸载方法2.
#删除nginx连带配置文件
sudo apt-get purge nginx # Removes everything.
#卸载不再需要的nginx依赖程序
sudo apt-get autoremove

备注：

  1. 查看window端口占用

netstat -ano
